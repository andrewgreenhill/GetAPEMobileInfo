Get APE Mobile Info: Documentation / Notes

Features

It handles more than 1000 items (overcoming the normal limit of the API).

Requirements

The Site needs to have "API enabled" setting turned on (under Admin => Organisation)

Templates

To understand the difference between Draft, Published and Archived templates, see https://apemobile.docs.apiary.io/#reference/forms => "Form collection"

The template_type output is re-mapped from numbers 1-5 to General Memo, Issue Memo, RFI Memo, Action and Form.

When exporting Published and Archived templates, their parent draft_template_id and draft_template_href is output into the CSV file.

Output

Output times are currently in 'Z' format.

How to get out unadjusted data in json format: change the output file extension from csv to json.
