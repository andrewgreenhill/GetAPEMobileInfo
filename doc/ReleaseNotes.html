Get APE Mobile Info: Release Notes

Version 0.6.5, Feb 2020

Improved appearance.

Fixed a bug in which the API's default limit of 100 records was sometimes not overridden.

Version 0.6.4, Feb 2020

Improved endpoint options display.

Added parameters for getting Forms.

Added ability to display a form sample PDF.

Increased robustness.

Version 0.6.3, Feb 2020

Added options for getting Actions, Forms, Memos and Punch Lists.

Simplified the default set of options by making some non-default additional options.

Version 0.6.2, Feb 2020

Added options for getting Project Members, Lists and WBS items.

Version 0.6.1, Jan 2020

Added options for getting Drawings & DocsViews and Annotations.

Version 0.6.0, Jan 2020

Re-mapped user_type output to up to date descriptions instead of the API's old/original terminology.

Version 0.5.9, Jan 2020

Changed template CSV output to:

  • Describe template_types as words instead of numbers (like Action instead of 4)
  • Output draft_template href and id (for published and archived templates)

Version 0.5.8, Jan 2020

Added options for modifying which info is requested for these types:

  • Users
  • Projects
  • Templates
  • Drawings & Documents

Version 0.5.1, Jan 2020

Made endpoint support extensible.

Added option to get Drawings & Docs.

Added ability to output to JSON format.
